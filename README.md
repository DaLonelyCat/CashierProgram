Cashier program made for my Current Business needs, Money management and expenses logging whille also being a typical cashier program

Features:
- 2 Language support (Indonesian and English)
- Profiles Support
- Location to save Logs
- Print receipts via checkout or History
- Summary of what product got sold the most
- Summary of Total Revenue and Expenses
- Expense Logging and editing
- Menu Customization
- Dark mode and Light mode
- Able to edit accent colors
- Vertical and Grid layout depending on menu amount
- Purchase history filter and summary according to filter
